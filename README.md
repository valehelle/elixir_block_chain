# ElixirBlockChain

A simple example for creating Blockchain in Elixir.

To run test
```
mix test
```

Run in console
```elixir
iex -S mix
blockchain = ElixirBlockChain.Blockchain.new()
blockchain = ElixirBlockChain.Blockchain.add_block(blockchain, "new Block added")
```
